# 🚀 TeamPulse

> **Feel the beat of your team.**

TeamPulse is a remote team connection and wellbeing platform that gives managers real-time visibility into how their team feels — and gives teammates a human way to stay connected, every single day.

## 🌐 Live Site

**[View Live on GitHub Pages →](https://srj245.github.io/Team-Pulse)**

## ✨ Features

- 🌡️ **Daily Pulse Check-ins** — 30-sec mood & energy check-ins with AI-analyzed trends
- 🤝 **Smart Connection Matching** — AI pairs teammates for optional virtual coffees
- 📊 **Team Energy Heatmap** — Real-time dashboard showing team morale over time
- 🎉 **Celebrations & Shoutouts** — Public appreciation feed
- 🔔 **Slack Integration** — Native bot for zero-friction check-ins
- 🤖 **Proactive AI Alerts** — Burnout detection before it happens

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3, Vanilla JS, Chart.js
- **Design:** Inter font, custom CSS animations, mobile-responsive
- **Branding:** Electric Violet `#6C63FF` + Coral Pink `#FF6584` + Mint `#43D9AD`
